О программе

Strata

Strata - это графический редактор с механизмом слоёв похожий на Adobe Photoshop. Слой - плоская двумерная растровая картинка, на которой можно рисовать мышкой. В тех местах, где явно ничего не нарисовано - слой считается прозрачным.

Редактор представляет из себя:

  1. "Рабочая область" - некоторое количество наложенных друг на друга прозрачных слоёв;
  2. Инструменты управления слоями:
    • Панель цвета карандаша (CTRL+SHIFT+C);
    • Панель толщины карандаша (CTRL+SHIFT+P);
    • Панель слоёв (CTRL+SHIFT+L);
    • Панель основных действий (CTRL+SHIFT+M);
  3. Рабочая область отображает слои в наложенном друг на друга виде;
  4. Операции со слоями:
    • Рисование в слой;
    • Добавление слоя;
    • Удаление слоя;
    • Перемещение слоёв;
    • Изменение порядка слоёв;
    • Скрытие/показ слоя;
  5. Проект, все слои проекта, сохраняются и загружаются в один файл в сериализованном виде:
    • Проект сохраняется в файл с названием "StrataProject" с раширением "*.st".
    • По умолчанию проект сохорянается в папку: %APPDATA%\Strata\Projects\

Просмотр ошибок

Просмотр ошибок осуществляется в Event Viewer. Event Viewer записывает любое действие, предпринятое на вашем компьютере. Вы можете использовать этот инструмент для просмотра ошибок программы.

Чтобы воспользоваться Event Viewer:

  1. В Windows откройте Панель управления > Администрирование > Event Viewer.
  2. Нажмите правой кнопкой мыши на ошибку Strata и выберите Свойства.
  3. Используйте поле Описание диалогового окна для определения ошибки.

Формат сохранения данных

Проект, все слои проекта, сохраняются и загружаются в один файл в сериализованном виде:

  1. Проект сохраняется в файл с названием "StrataProject" с раширением "*.st".
  2. По умолчанию проект сохорянается в папку: %APPDATA%\Strata\Projects\

Исходное изображение сохраняется в папку "Мои дкументы" в файл с доступными рашиениями:

  1. *.png
  2. *.jpg
  3. *.gif
